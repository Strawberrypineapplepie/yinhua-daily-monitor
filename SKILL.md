# 银华日利套利监控器

## 📌 监控标的

- **基金名称**: 银华日利货币市场基金
- **场内代码**: 511880 (上海)
- **最小单位**: 1手 = 100份
- **每份面值**: 约100元
- **日收益**: 万0.3-0.4 (速算用万0.35)

## 📊 核心策略 (基于用户文档)

### 1. 实际价值计算公式

| 日期类型 | 公式 | 示例 |
|---------|------|------|
| 正常工作日 | 实际价值 = 昨日净值 + 0.0035 | 100.3762 + 0.0035 = 100.3797 |
| 周五晚上 | 实际价值 = 昨日净值 + 0.0035 × 3 | 含周末收益 |
| 节假日(n天) | 实际价值 = 昨日净值 + 0.0035 × (n+1) | 清明3天: +0.0035×4 |

### 2. 套利策略模式

#### 模式A: 折价买入赎回套利 ⭐ (几乎无风险)
- **触发条件**: 折价 ≥ 万0.8
- **操作**: 场内折价买入 → 赎回
- **收益**: 折价差额
- **优势**: 几乎无风险，T+1盘中到账
- **注意**: 推荐某银某河券商(T+1盘中到账)

#### 模式B: 溢价申购卖出套利 ⭐⭐⭐ (中低风险)
- **触发条件**: 溢价 ≥ 万1.5
- **操作**: 尾盘申购 → T+2到账卖出
- **风险**: T+2价格波动风险
- **建议**: 仅在明显溢价时操作

#### 模式C: 日内做T ⭐⭐ (低风险)
- **触发条件**: 溢价 ≤ 万0.5
- **操作**: 低价买入 → 高价卖出
- **技巧**: 结合IOPV判断

#### 模式D: 周二特殊策略 ⭐⭐ (中低风险)
- **触发条件**: 周二溢价 ≥ 万1
- **操作**: 周二溢价申购 → 周四卖出 + 逆回购
- **优势**: 避免周末风险

### 3. 风险提示

- 单日巨额赎回可能导致净值异常
- 节假日前后收益计算需注意
- 套利空间过小时需考虑交易成本

## 🔧 配置要求

### 环境变量
- `MX_APIKEY`: 东方财富妙想API密钥
- `FEISHU_WEBHOOK`: 飞书机器人Webhook地址

### 数据源
1. 主数据源: 东方财富妙想技能 (mx-data)
2. 备用数据源: 东方财富网行情接口

## 📤 飞书通知格式

每次检测结果将包含:
- 当前价格、IOPV净值
- 折溢价幅度及年化收益
- 策略建议(含具体操作)
- 风险提示

## 使用方法

```bash
# 手动运行一次检测
python scripts/yinhua_monitor.py --once

# 启动定时监控 (开市时间每10分钟)
python scripts/yinhua_monitor.py --schedule

# 仅获取当前数据不推送
python scripts/yinhua_monitor.py --fetch-only
```
